# Python Intern Task

This project demonstrates how to fetch data from a public API using Python's `requests` library, handle JSON data, and display user information in a readable format.

## Task Description
- Fetch user data from: `https://jsonplaceholder.typicode.com/users`
- Display the following details for each user:
  - Name
  - Username
  - Email
  - City (from `address.city`)
- Bonus:
  - Print only users whose city name starts with the letter **S**.
  - Handle API errors gracefully.

##  How to Run the Script

1. **Clone the repository**
   git clone https://github.com/Mridulsharma1133/Python-Intern-Task.git
   cd Python-Intern-Task

2. **Install dependencies**
   pip install requests

3. **Run the script**
   python script.py

**Example Output**
User 1:
Name: Leanne Graham
Username: Bret
Email: Sincere@april.biz
City: Gwenborough
